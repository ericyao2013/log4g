Log4gSimpleLayout

Log4gSimpleLayout — A simple log layout

Synopsis

                    Log4gSimpleLayout;
                    Log4gSimpleLayoutClass;

Description

The simple layout consists of the level of a log statement and the log message itself separated by " - ". For example: 

DEBUG - Hello, World!

Details
