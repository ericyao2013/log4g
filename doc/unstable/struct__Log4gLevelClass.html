Log4gLevel* _Log4gLevelClass::ALL

The ALL level has the lowest possible rank and is intended to turn on all logging.

Log4gLevel* _Log4gLevelClass::DEBUG

The DEBUG level indicates fine-grained informational events that are useful for debugging.

Log4gLevel* _Log4gLevelClass::ERROR

The ERROR level indicates an error event that may allow the application to continue.

Log4gLevel* _Log4gLevelClass::FATAL

The FATAL level indicates a very severe error events that will presumably lead the application to abort.

Log4gLevel* _Log4gLevelClass::INFO

The INFO level indicates informational messages that highlight the progress of the application at a coarse-grained level.
