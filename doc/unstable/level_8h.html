Detailed Description

Minimum set of logging levels recognized by the system.

Author:
Mike Steinert
Date:
1-29-2010

The following base log levels are defined:

  1. OFF
  2. FATAL
  3. ERROR
  4. WARN
  5. INFO
  6. DEBUG
  7. ALL

The Log4gLevel class may be sub-classed to define a larger level set.
