Detailed Description

Disable logging.

Author:
Mike Steinert
Date:
2-22-2010

Including this file instead of log4g/log4g.h will disable all logging in your application. All functions and macros defined in log4g/log4g.h, log4g/mdc.h, and log4g/ndc.h will be disabled. If you are using other functions from the log4g package then you may disable them by enclosing them in the LOG4G_DISABLE macro.
