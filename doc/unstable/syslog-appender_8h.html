Detailed Description

Append to the local syslog.

Author:
Mike Steinert
Date:
2-11-2010

This is a simple syslog appender that appends to the local syslog using the syslog(3) system call.

Note:
This is a very basic implementation and could use some work.
See also:
syslog(3)
