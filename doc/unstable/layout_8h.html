Detailed Description

Log event layout base class.

Author:
Mike Steinert
Date:
2-5-2010

Users may extend this class to implement custom log event layouts.

Many appenders require a layout in order to log an event. Sub-classes must override the _Log4gLayoutClass::format() virtual function to implement custom formatting.
