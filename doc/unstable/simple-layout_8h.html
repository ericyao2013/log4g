Detailed Description

A simple log layout.

Author:
Mike Steinert
Date:
2-5-2010

The simple layout consists of the level of a log statement and the log message itself separated by " - ". For example:

 DEBUG - Hello, World!
See also:
log4g/layout/pattern-layout.h
