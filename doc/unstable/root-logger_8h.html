Detailed Description

The root logger.

Author:
Mike Steinert
Date:
2-10-2010

The root logger sits on top of the logger hierarchy. It is a normal logger with the following exceptions:

  1. It may not be assigned a NULL level threshold
  2. The root logger cannot have a parent (the get_effective_threshold() method always returns the value of the level field without walking the hierarchy)
