Detailed Description

Internal logging API.

Author:
Mike Steinert
Date:
3-10-2010

This class is used to output log statements from within the Log4g package.

Log4g components cannot use the Log4g API. This API allows Log4g to output debugging statements to stdout. To enable debugging define the debug flag (Bash): 

 export LOG4G_FLAGS=debug

All internal debug statements go to stdout, error and warn messages are sent to stderr. All internal messages are prepended with the string "log4g: ".
